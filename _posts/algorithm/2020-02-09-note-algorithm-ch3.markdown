### 渐进记号
  渐进准确界 Θ(g(n)) : $ 0 <= c_1 g(n) <= f(n) <= c_2g(n) $    
  渐进上界 O(g(n)) : $ 0 <= f(n) <= cg(n) $    
  非渐进准确上界 o(g(n)): $ 0 <= f(n) < cg(n) $    
  渐进下界 Ω(g(n)) : $ 0 <= cg(n) <= f(n) $    
  非渐进准确下界 ω(g(n)) : $ 0 <= cg(n) < f(n) $
